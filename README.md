### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation & Setup

1. Clone the repository
```bash
git clone https://github.com/Tejas1024/Tejas1024.github.io.git
cd Tejas1024.github.io
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Start development server
```bash
npm run dev
# or
yarn dev
```

4. Build for production
```bash
npm run build
# or
yarn build
```

5. Preview production build
```bash
npm run preview
# or
yarn preview
```
